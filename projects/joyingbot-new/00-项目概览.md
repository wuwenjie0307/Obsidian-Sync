# joyingbot-new

## 基本信息

- 路径: `C:\Users\admin\Desktop\joyingbot-new`
- 分支: `test`
- 类型: Python 后端服务

## 核心模块

| 目录 | 用途 |
|---|---|
| `router/` | HTTP/API 路由层与业务入口 |
| `router/service/video_server2/` | H20 视频生成、VoxCPM、LatentSync、字幕、音频处理主链路 |
| `agent/hyper_agent/` | Hyper Agent 对话、链路与节点逻辑 |
| `scheduler/` | 定时任务、视频任务扫描与资源池调度 |
| `crm/` | CRM 请求、签名、标签与客户数据相关工具 |
| `common/` | 公共客户端、缓存、日志、数据库与文本工具 |
| `config/` | 本地、测试、生产配置文件 |
| `deploy/` | H20 Docker 与部署说明 |
| `test/` | 回归测试与业务链路测试 |

## 配置

- `config/config-dev.json` — 开发/测试配置
- `config/config-local.json` — 本地配置
- `config/config-pro.json` — 生产配置
- `requirements.txt` — Python 依赖
