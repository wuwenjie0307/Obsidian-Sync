# 模型更换方案

## 目标

替换两个模型，部署到服务器本地：

| 替换什么 | 现用方案 | 替换为 |
|----------|----------|--------|
| 唇形同步（数字人视频） | guiji2025/duix.avatar:2.9（Docker） | [LatentSync](https://github.com/bytedance/LatentSync)（字节跳动） |
| 音色克隆 / TTS | 阿里云 DashScope cosyvoice-v3-plus（API） | [VoxCPM](https://github.com/OpenBMB/VoxCPM)（清华/面壁） |

## 新模型信息

### LatentSync — 唇形同步

- 用途：数字人视频生成、口型同步
- 显存：8 GB（v1.5）/ 18 GB（v1.6）
- Python：3.10
- CUDA：12.1+
- 安装：git clone + `source setup_env.sh`
- 模型权重：~7 GB（HuggingFace 下载）
- 许可证：开源

### VoxCPM — 音色克隆 + TTS

- 用途：零样本语音克隆、文本转语音
- 显存：6-8 GB
- Python：3.10+
- CUDA：12.0+
- 安装：`pip install voxcpm`
- 模型权重：自动下载
- 许可证：Apache 2.0（可商用）
