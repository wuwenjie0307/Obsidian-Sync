## VS Code Remote-SSH 连接

### 1. 安装插件

在 VS Code 扩展商店搜索 **Remote - SSH**（微软官方），安装。

### 2. SSH 配置

配置文件路径：`C:\Users\admin\.ssh\config`

```
Host joying-server
    HostName 222.71.55.27
    Port 9527
    User developer
```

### 3. 连接步骤

1. 按 `F1` → 输入 **Remote-SSH: Connect to Host**
2. 选择 `joying-server`
3. 等待连接完成，左下角出现绿色 `SSH: joying-server` 即成功
4. 点"打开文件夹"浏览服务器上的项目目录

### 4. Xshell 备用连接

如果 VS Code SSH 不可用，可用 Xshell 连接：
- 主机：`222.71.55.27`
- 端口：`9527`

- **运行时间**：629+ 天（上次重启约 2024年9月）

## 注意事项

- 服务器多人共用，操作前用 `w` 命令查看在线用户
- 不建议随意重启或关机
- Docker 操作需 `sudo`
- 退格键问题：Xshell 属性 → 终端 → 键盘 → Backspace 设为 `ASCII 127 (DEL)`
