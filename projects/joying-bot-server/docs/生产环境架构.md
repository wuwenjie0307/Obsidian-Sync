### h20 全链路架构（测试服模式）

```
h20 (localhost)
├─ bot 主服务（Flask/调度/定时任务）
├─ VoxCPM TTS API         → localhost:8100
├─ LatentSync 唇形 API    → localhost:8101
├─ Whisper 语音识别        → 本地进程池
├─ ffmpeg 视频处理         → 本地
├─ 文件上传               → 本地

└─ Redis                  → 222.71.55.27:6389（远程）
```

所有模型调用都是 localhost，不存在跨服务器网络问题。

### h20 服务启动命令（h20 重启后执行）

所有服务 nohup 后台运行，supervisor 待晋良配置。

**1. VoxCPM API（base 环境）**
```bash
nohup python /data/projects/joyingbot-new/router/service/video_server/voxcpm_api.py --port 8100 > /tmp/voxcpm.log 2>&1 &
```

**2. LatentSync API（latentsync 环境，需要 LD_LIBRARY_PATH）**
```bash
export LD_LIBRARY_PATH=/usr/local/cuda-12.4/targets/x86_64-linux/lib:$LD_LIBRARY_PATH
nohup conda run -n latentsync --no-capture-output python /data/projects/joyingbot-new/router/service/video_server/latentsync_api.py --port 8101 > /tmp/latentsync.log 2>&1 &
```

**3. Bot 主服务（joyingbot 环境，需要 TZ=Asia/Shanghai + 连测试库）**
```bash
cd /data/projects/joyingbot-new && TZ=Asia/Shanghai nohup conda run -n joyingbot --no-capture-output python app_server_api.py --env dev > /tmp/bot_dev.log 2>&1 &
```

**验证**
```bash
curl -s http://localhost:8100/health && echo " VoxCPM OK"
curl -s http://localhost:8101/health && echo " LatentSync OK"
curl -s -X POST http://localhost:8018/crm/voice_clone_audition \
  -H "Content-Type: application/json" \
  -d '{"voice_file_url":"http://localhost:9998/voice.mp3","text":"测试"}' && echo " Bot OK"
```

⚠️ 启动顺序必须先 VoxCPM + LatentSync，最后 Bot。Bot 启动约 30s（加载 Whisper 模型）。

## 生产 video_work_Heygem_Whisper 函数签名（2026-05-28 审查）

```python
def video_work_Heygem_Whisper(
    Original_video_url: str,       # duix.avatar Docker 地址
    Original_audio_url: str,       # 参考音频 URL
    Video_url: str,                # 用户上传视频 URL
    Original_Creation_text: str,   # TTS 文案
    task_id: str = None,
    Montage_dict: dict = None,     # {video_url: description}
    Montage_dict_bool: dict = None,# {video_url: True/False} 是否需要唇形同步
    Photo_dict: dict = None,       # {photo_url: description}
    background_music_url: str = None,
    First_frame_judge_url: str = None,
    model_name: str = "medium",    # Whisper 模型
) -> tuple[str, float]:           # (output_video_url, duration)
```

⚠️ **没有 `voice_emotion`、`voice_speed`、`voice_volume` 参数** — 切换 VoxCPM 时需要新增。

## h20 API 并发保护（2026-05-28 新增）

为防止多任务并发时 GPU OOM，在两个 API 中加入了 `threading.BoundedSemaphore`：

| 服务 | 并发数 | 超时 | 行为 |
|------|--------|------|------|
| VoxCPM (8100) | 2 | 300s | 第 3 个请求排队等，超时返回 503 |
| LatentSync (8101) | 1 | 600s | 后续请求排队等，超时返回 503 |

修改文件：
- `router/service/video_server/voxcpm_api.py`
- `router/service/video_server/latentsync_api.py`

已上传到 h20 并重启服务，新 PID 待确认。
