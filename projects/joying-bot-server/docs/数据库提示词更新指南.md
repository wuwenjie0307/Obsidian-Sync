# 数据库提示词更新指南

## 背景

AI 提示词（如客户激活文案、楼市周报等）存储在 MySQL 数据库表 `t_agent_prompt_config` 中，不在代码文件里。代码通过 `agent_config_service.get_prompt()` 从数据库读取，并缓存到 Redis。

## 关键代码位置

| 文件 | 作用 |
|------|------|
| [common/agent_const.py](common/agent_const.py#L272) | 定义 `AI_CLIENT_ACTIVATE` 等常量 |
| [router/crm_server.py](router/crm_server.py#L1681) | 使用 `get_prompt(COMMON, AI_CLIENT_ACTIVATE)` |
| [router/service/agent_config_service.py](router/service/agent_config_service.py) | 从 DB 读 prompt，缓存到 Redis（1小时） |
| [pojo/models.py](pojo/models.py#L312) | `AgentPromptConfig` 模型，表名 `t_agent_prompt_config` |

## 更新步骤

### 1. 找到要改的提示词

先在代码里用全局搜索（如搜"客户激活"）找到对应的常量名：

```
common/agent_const.py → AI_CLIENT_ACTIVATE = "AI_CLIENT_ACTIVATE"
```

### 2. 查询数据库

```sql
USE zhugedata;

SELECT id, model, type, LEFT(prompt, 100) 
FROM t_agent_prompt_config 
WHERE model = 'COMMON' AND type = 'AI_CLIENT_ACTIVATE';
```

记下 `id`（如 149）。

### 3. 执行更新

```sql
UPDATE t_agent_prompt_config 
SET prompt = '新提示词内容...' 
WHERE id = 149;
```

**注意：**
- 如果提示词里有单引号 `'`，需要转义成 `''`（两个单引号）
- 建议先 `SET prompt = '测试'` 测试一下 UPDATE 是否能正常执行，再替换成正式内容

### 4. 验证

```sql
SELECT prompt FROM t_agent_prompt_config WHERE id = 149;
```

对比关键差异点，确认更新成功。

### 5. Redis 缓存处理

- **不处理**：等待最多 1 小时，缓存自动过期，下次读取从 DB 获取新内容
- **立刻生效**：连接到 Redis 清除缓存 key

```bash
redis-cli -p 6389 -a 123456 DEL prompt:COMMON:AI_CLIENT_ACTIVATE
```

缓存 key 格式：`prompt:{model}:{type}`

## 常见错误及解决

### 错误 1：VSCode 插件双击编辑报"没有主键"

**现象**：在查询结果表格里双击单元格编辑，Ctrl+S 保存时报"没有主键，此改动可能会比预想的更多"

**原因**：VSCode MySQL 插件对某些查询结果无法识别主键列，拒绝直接编辑

**解决**：使用 SQL UPDATE 语句，不要用表格编辑功能

### 错误 2：UPDATE 执行很久不结束

**现象**：执行 UPDATE 后一直转圈，看起来卡住了

**排查步骤**：
```sql
SHOW PROCESSLIST;
```

- 如果所有连接都是 `Sleep` 状态 → UPDATE 其实已经执行完了，是插件没刷新
- 点红色停止按钮取消，直接跑 SELECT 验证
- 如果有非 Sleep 的连接长时间运行 → 可能被锁，取 `Id` 执行 `KILL <Id>`

### 错误 3：UPDATE 执行完但内容没变

**现象**：跑了 UPDATE，SELECT 出来还是旧的

**可能原因**：
1. 插件开了多个查询窗口，旧查询窗口未关闭导致事务未提交
2. 内容中有特殊字符（如单引号）导致 SQL 语法错误但没报错

**解决**：
1. 关闭所有查询窗口，新建一个查询窗口重新执行
2. 先跑 `UPDATE ... SET prompt = '测试' WHERE id = 149;` 确认基础 UPDATE 可用
3. 再跑正式的内容更新
