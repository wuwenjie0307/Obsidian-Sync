### 踩坑记录

| 问题                              | 原因                                                | 解决                                                                               |
| ------------------------------- | ------------------------------------------------- | -------------------------------------------------------------------------------- |
| GPU 不可用，回退到 CPU                 | pip 自动装的 PyTorch 2.12 需要 CUDA 12.8，但服务器驱动只支持 12.4 | 降级：`pip install torch==2.5.1 --index-url https://download.pytorch.org/whl/cu124` |
| torchaudio 报 libcudart.so.13 缺失 | torchaudio 2.11 与 torch 2.5 不兼容                   | 重装 torchaudio（pip 自动匹配版本）                                                        |
| 生成音频 4 秒                        | 正常，文本短对应音频短                                       | —                                                                                |

### CLI 使用

```bash
# 先连接 h20（从生产服务器跳板：sudo su → ssh root@h20 -p 10019）

# 基础合成
voxcpm --text "要合成的文本" --output /tmp/output.wav

# 语音克隆（需要参考音频）
voxcpm --text "要合成的内容" \
  --prompt-audio /path/to/voice.wav \
  --prompt-text "参考音频对应的文本" \
  --output /tmp/output.wav
```

### Python API

```python
from voxcpm import VoxCPM
model = VoxCPM.from_pretrained("openbmb/VoxCPM2")
wav = model.generate(text="你好世界", prompt_wav_path="speaker.wav", prompt_text=None)
```

### 安装步骤

```bash
# 1. 创建 conda 环境
conda create -n latentsync python=3.10.13 -y
conda activate latentsync

# 2. 安装 ffmpeg
conda install -y -c conda-forge ffmpeg

# 3. 安装系统依赖
sudo apt -y install libgl1

# 4. 安装 Python 依赖
cd /root/LatentSync
pip install -r requirements.txt

# 5. 下载模型权重（国内用镜像加速）
export HF_ENDPOINT=https://hf-mirror.com
huggingface-cli download ByteDance/LatentSync-1.6 whisper/tiny.pt --local-dir checkpoints
huggingface-cli download ByteDance/LatentSync-1.6 latentsync_unet.pt --local-dir checkpoints
```

## CRM 接口参数映射

### 短视频任务列表返回的 voice 字段 → VoxCPM API

| CRM 字段 | VoxCPM API 参数 | 类型 | 可选值 |
|----------|----------------|------|--------|
| `voice_emotion` | `voice_emotion` | int (1-8) | 1=正常 / 2=亲切 / 3=热情 / 4=激昂 / 5=严肃 / 6=喜悦 / 7=悲伤 / 8=愤怒 |
| `voice_speed` | `voice_speed` | float | 0.75 / 1.0 / 1.25 / 1.5 / 2.0 / 3.0（默认 1.0） |
| `voice_volume` | `voice_volume` | int | 0-100（默认 50） |

### voice_emotion 枚举（CRM 传整数，API 同时支持 int/string）

| 值 | 中文 | API 可接受 |
|----|------|-----------|
| 1 | 正常 | `1` / `"1"` / `"normal"` |
| 2 | 亲切 | `2` / `"2"` / `"亲切"` |
| 3 | 热情 | `3` / `"3"` / `"热情"` |
| 4 | 激昂 | `4` / `"4"` / `"激昂"` |
| 5 | 严肃 | `5` / `"5"` / `"严肃"` |
| 6 | 喜悦 | `6` / `"6"` / `"喜悦"` |
| 7 | 悲伤 | `7` / `"7"` / `"悲伤"` |
| 8 | 愤怒 | `8` / `"8"` / `"愤怒"` |

### 接口路径

- **列表接口**: `POST /csm/agent/pc/video/generateTaskList`
- **提交接口**: `POST /submit_heygem_whisper_video_task`
- **试听接口**: `POST /crm/voice_clone_audition`（直接调 h20 VoxCPM API）

### voice_speed 注意事项

- CRM 正式规范**不允许 0.5**（仅 0.75~3.0），VoxCPM API 的 `ALLOWED_SPEEDS` 已对齐
- CRM 返回数据中曾出现过 `0.5`（可能是旧数据），代码以 CRM 正式规范为准

### 数据流（当前 vs 期望）

```
当前（断层）：
CRM任务数据(含voice字段) → submit路由(未提取) → video_work(未接收) → clone函数(硬编码) → API

期望：
CRM任务数据(含voice字段) → submit路由(提取) → video_work(转发) → clone函数(传参) → API
```

### 额外缺失字段：reference_text

- CRM 没有单独字段，但 video_work 中 `simplified_text`（Whisper 转录）可作为值

## 视频生成完整数据流

### 两阶段流程

```
┌─────────── CRM 提交任务 ───────────────────┐
│ voice_file_url       → 参考音色             │
│ original_creation_text → 要念的文案          │
│ original_video_url   → 数字人底板视频        │
│ voice_emotion/speed/volume → TTS 参数       │
└────────────────┬─────────────────────────────┘
                 ↓
┌─────────── 阶段1: 声音克隆 (VoxCPM) ───────┐
│ 服务: voxcpm_api.py :8100                   │
│ 输入: 参考音频URL + 文案 + emotion/speed/vol │
│ 输出: 合成音频 WAV (48kHz)                   │
│ 耗时: ~10s                                  │
└────────────────┬─────────────────────────────┘
                 ↓  合成音频 URL
┌─────────── 阶段2: 唇形同步 (LatentSync) ───┐
│ 服务: latentsync_api.py :8101               │
│ 输入: 数字人底板视频URL + 合成音频URL        │
│ 输出: 嘴型对齐最终视频 MP4                   │
│ 耗时: ~90s                                  │
└────────────────┬─────────────────────────────┘
                 ↓
           最终数字人视频
```

### 参数传递链路

```
crm_server.py submit 路由
  ├─ 提取 voice_emotion/speed/volume 字段
  ├─ 构建 payload
  └─ ↓
video_work_Heygem_Whisper()
  ├─ 接收 voice_emotion/speed/volume（默认值 normal/1.0/50）
  ├─ ↓
  ├─ clone_voice_and_synthesize(audio_url, text, task_id,
  │       reference_text=simplified_text,    ← Whisper 转录文本
  │       voice_emotion=voice_emotion,
  │       voice_speed=voice_speed,
  │       voice_volume=voice_volume)
  │   → VoxCPM API (8100) → 合成音频
  ├─ ↓
  └─ LatentSyncService.generate_video(audio_url, video_url, task_id)
      → LatentSync API (8101) → 唇形同步视频
```
