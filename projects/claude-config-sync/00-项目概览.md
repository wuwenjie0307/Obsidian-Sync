# claude-config-sync

## 基本信息

- 路径: `c:\Users\admin\Desktop\claude-config-sync`
- 分支: `main`
- 类型: 配置管理
- 仓库: https://github.com/wuwenjie0307/claude-config-sync

## 用途

一键同步 Claude Code 的 skills、agents、hooks、MCP servers 和 plugin 配置到新设备。

## 核心模块

| 目录/文件 | 用途 |
|---|---|
| `skills/` | 自定义 skill 文件（obsidian, mysql-query） |
| `config/` | 模板配置（mcp.json, hooks.json, settings） |
| `install.ps1` | Windows 一键安装脚本 |
| `install.sh` | macOS/Linux 一键安装脚本 |
| `README.md` | 完整清单和使用说明 |

## 同步的内容

- **自定义 Skills**: obsidian, mysql-query
- **Plugins**: Understand-Anything（8 skills, 9 agents）
- **MCP Servers**: api-lab, brave-search, codegraph
- **Hooks**: Understand-Anything 自动更新
