## 踩坑记录

- VSCode MySQL 插件双击编辑表格会报"没有主键"，不能这么用，必须写 SQL
- UPDATE 执行后插件一直转圈但实际已完成，用 `SHOW PROCESSLIST` 确认
- 视觉模型 max_tokens 默认 2048 对大图片不够用，字段列表会被截断
- 操作前先确认仓库 — joying-bot-server（旧）和 joyingbot-new（新）容易混淆
- **h20 架构理解纠正**：h20 是完整测试服，bot 全套跑在本地，VoxCPM/LatentSync 都是 localhost 调用，不是纯 GPU 节点。生产部署后两服务器无任何交互，之前"生产→h20 网络需打通"是误判
- **LatentSync Face not detected**：InsightFace buffalo_l 对某些视频人脸检测不稳定，需用清晰正面人脸视频
- **first_frame_judge_url 要图片不是视频**：ffmpeg `-loop 1 -i video.mp4` 会报 Option not found
- **h20 时区冲突**：`/etc/timezone: Etc/UTC` vs `/etc/sysconfig/clock: Asia/Shanghai` → APScheduler 报 ZoneInfoNotFoundError，设 `TZ=Asia/Shanghai` 解决
- **LatentSync libnvrtc.so.12**：即使 conda 环境里 onnxruntime-gpu 装了，LD_LIBRARY_PATH 不设也会报 CUDA provider 不可用
- **ncpm run 嵌套 conda run** 可以工作，但 LD_LIBRARY_PATH 需在启动 API 时就设好
- **git.joyingai.cn DNS**：本机和 h20 都解析不了，需在内网或 VPN 环境才能访问。已在内网环境下 push 成功
